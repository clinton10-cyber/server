Database

This app uses a single DATABASE_URL environment variable for its Postgres connection — it works identically whether that URL points to a plain Postgres instance (e.g. on Render) or a Supabase Postgres connection string (found in your Supabase project under Settings → Database → Connection string). No code changes are needed to switch between them, just update DATABASE_URL and restart.
